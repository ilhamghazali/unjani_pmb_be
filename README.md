# unjani_pmb_be

Backend PMB Unjani

## Development
1. Clone repo
```bash
git clone https://gitlab.com/unjani/unjani_pmb_be
```

2. cd to folder
```bash
cd unjani_pmb_be
```

3. start server
```bash
npm start
```

## Sonarqube
Please change the sonar.projectKey value in ``sonar-project.properties`` according to the project in SonarQube. As an example:
```
sonar.projectKey=unjani-api-auth
sonar.sources=./
```
